**Basic Features**

* As a **new user**, I want to create an account (sign up) so that I can save my zoo visits and sightings.
* As a **logged-in user**, I want to log that I visited Zoo X on DATE so that I can track which zoos I’ve been to.
* As a **user**, I want to search for a zoo by name or location so that I can find the zoo I want to log or explore.
* As a **user**, I want to view a zoo’s detail page (name, address, map) so that I can learn about each zoo before I visit.
* As a **user**, I want to record that I saw Animal Y at Zoo X at TIME, with an optional photo and note, so that I can document my sightings.
* As a **user**, I want to search for an animal by name so that I can find its profile and see where it’s available.
* As a **user**, I want to view an animal’s detail page (photo, description, list of zoos) so that I can learn about the species.
* As an **admin** I want to add new animals and create animal detail pages.
* As an admin I want to add new zoos.
* As an admin I want to add new animal to the inventory of a zoo.
* As an admin I want to delete an animal from the inventory of a zoo.
* An an admin I want to edit the detail page of an animal.

---

**Intermediate Features**

* As a **user**, I want to view a list of all my zoo visits and animal sightings so that I can review my history.
* As a **user**, I want to edit a log of a previous visit or sighting (change date, add/remove photo, update note) so that my records stay accurate.
* As a **user**, I want to delete a visit or sighting log so that I can remove mistakes or unwanted entries.
* As a **user**, I want to delete a zoo visit log so that I can remove mistakes or unwanted entries.
*  As a **user**, I want to delete an animal sighting log so that I can remove mistakes or unwanted entries.
* As a **user**, I want to see which animals I’ve already seen and which I haven’t so that I know my progress.


---

**Advanced Features**

* As a **user**, I want to earn badges (e.g. “20 zoos visited,” “100 birds seen”) so that I’m motivated to continue exploring.
* As a **user**, I want to find the nearest zoo that has an animal I haven’t seen so that I can plan my next visit.
* As a **user**, I want to view a map with markers for zoos (filtered by those that have selected animals) so that I can visualize nearby opportunities.
* As a **user**, I want to see recommendations for animals I haven’t seen yet that are common in my area so that I can easily expand my collection.
* As a **user**, I want to follow other users and see their public logs so that I can discover new zoos and animals through friends.
* As a **user**, I want to share my visit or sighting (with photo) on social media so that I can show my achievements to others.
* As a **power user**, I want to suggest a new zoo or add an animal to a zoo’s inventory (subject to moderation) so that the database stays up to date.
* As an **admin**, I want to review and approve user-submitted zoos and animal entries so that the platform data remains accurate.
* As a **user**, I want to receive notifications when I unlock a badge or when a zoo near me adds a new species so that I stay engaged.
